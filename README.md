## Fundamental

[HTTP State Management Mechanism](https://tools.ietf.org/html/rfc6265)

## Prerequisites

MUST run the following command to create local sll certification which is used to https server.

```bash
npm run setup
```

- start https server

```bash
npm run start-https
```
